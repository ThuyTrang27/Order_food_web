# 🍔 Order-food-web

Dự án website đặt món ăn trực tuyến được xây dựng bằng **React**, giúp người dùng dễ dàng tìm kiếm, lựa chọn và quản lý giỏ hàng thực phẩm một cách nhanh chóng.

---

## 🚀 Tính năng chính (Main Features)

* **Danh sách món ăn:** Hiển thị thực đơn đa dạng với hình ảnh và giá cả rõ ràng.
* **Giỏ hàng (Cart):** Thêm/bớt món ăn, tự động tính tổng tiền trong thời gian thực.
* **Tìm kiếm:** Lọc nhanh các món ăn yêu thích theo tên.
* **Responsive Design:** Giao diện hiển thị tốt trên cả máy tính và điện thoại.

---

## 🛠 Công nghệ sử dụng (Technologies Used)

### 💻 Frontend
* **React.js**: Sử dụng Hooks (`useState`, `useEffect`) để quản lý trạng thái ứng dụng.
* **Styling**: CSS3 / Styled-components (Tạo giao diện hiện đại, dễ tùy chỉnh).

### ⚙️ Hệ thống & Dữ liệu
* **Dữ liệu**: Fetch API / Mock data để mô phỏng luồng dữ liệu thực tế.
* **Công cụ**: Vite / Create React App, Git & GitHub.
